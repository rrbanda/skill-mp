# Skills Marketplace

An open-source platform for discovering, building, and managing AI agent skills.

## What is Skills Marketplace?

Skills Marketplace provides a unified format (`SKILL.md`) for defining AI agent prompt instructions that work across platforms like Cursor, Claude Code, Windsurf, Google ADK, and more.

**Key features:**

- **Browse and search** 180+ production-grade skills organized by category
- **Interactive knowledge graph** powered by Neo4j for exploring skill relationships
- **AI-powered skill builder** that generates SKILL.md files from natural language
- **Cross-platform export** to Cursor, Claude Code, Windsurf, ADK, and more
- **GraphRAG pipeline** for intelligent relationship discovery between skills
